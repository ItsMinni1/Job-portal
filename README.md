This Job-portal application is designed to simulate a real-world job search and recruitment system. Users can register as job seekers or employers, log in securely, and perform role-based actions. Employers can create, manage, and review job postings. Job seekers can search jobs, apply to listings, and view application status. Administrative controls are supported where applicable. Testing: Unit / integration tests

**Installation:**
git clone https://github.com/ItsMinni1/Job-portal.git
cd Job-portal

**Install dependencies:**
Server:
cd server
npm install
Client:
cd ../client
npm install

**Usage**
Start backend server:
cd server
npm start
start frontend development server:
cd client
npm start

Open browser and navigate to http://localhost:port_number

To run tests, use "npm test" in both directories.

**This project is open source.**
